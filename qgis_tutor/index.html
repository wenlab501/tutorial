QGIS 人口地圖繪製教學網

目的

製作這個網站的主要目的是為了由台大人口中心與台大地理系主辦，溫在弘教授主講的「人口數據科學」工作坊：QGIS的人口地圖繪製 (2018/1/15～16 )密集課程提供教學材料而準備。課程授課以溫教授的教學內容為主，這網站的主要功能是供學員自行查詢與參考。

課程主軸在於繪製各種以人口為主的主題地圖，因此網站內容講著重於相關統計資訊的圖表為主，其他 QGIS 操作、分析功能將視情況補充。

此教學網站的主要章節如下：

  • 入門基礎
  • 介面介紹
  • 基本地圖繪製
  • 統計地圖繪製
  • 進階地圖繪製
  • 圖資處理

什麼是QGIS

QGIS ，原名 Quantum GIS，是一個自由的 GIS(地理資訊系統) 軟體。其主要功能在於操作、分析空間資料，並提供繪製地圖的環境。

QGIS 與 Google Earth (GE) / Google Map (GM) 等軟體、平臺最大的差別在於：

  • QGIS 提供各種資料處理、查詢、分析功能，GE/GM 不提供；
  • GE/GM 提供的是一顆球 (地球) 與一個地圖環境 (web-mercator投影地圖)；QGIS 會自動根據圖層座標系統來設定投影環境，同時也讓使用者彈性的自由設定；
  • GE/GM 使用 kml/kmz 資料格式；絕大部分空間資料格式，包括shapefile、kml/kmz、有空間資訊欄位的 csv ，也可以從資料庫(postgis之類) 輸入檔案；
