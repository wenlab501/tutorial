面資料找中心點(centroid)

找中心點是很常會用到的一個動作，比方在教學中的泡泡圖，就需要對鄉鎮資料去找到中心點來繪製。 在 QGIS 中找面資料的幾何中心點非常簡單，過程如下所述：

以下主要用鄉鎮人口資料(點擊下載)作說明。

在 Vector-->Geometry Tools-->Polygon centroids，打開這工具的視窗：


這是幾何中心的視窗，只有兩個地方需要設定，第一個是 Input layer，即輸入的 面圖層；另一個是中心點圖層的儲存路徑與檔名，完成設定後點 Run 執行：


幾何中心點 centroid 就完成了：
