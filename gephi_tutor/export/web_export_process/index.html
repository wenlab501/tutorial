網頁輸出過程

製作中

安裝 SigmaExporter plugin

從 Gephi 輸出

因爲 網頁的座標軸部分，Y軸是從上往下增加的，與一般地圖的方向相反，所以需要先將 Y 座標做一次上下轉換，即乘上負號，產生新的欄位。

geo layout 設定的時候選對調後的 Y 欄位

儲存圖檔

File --> Export --> Sigma js templete... (安裝 上述提到的 plugin 後會出現的選項)

做一些基本設定

選擇儲存位置

完成

後續的步驟只是讓電腦可以開一個暫時的 server。若要上傳到伺服器 (已設定好 apache/nginx/lighttpd 之類的，如 臺大 homepage 空間)，則可以直接將這些輸出好的檔案上傳。

設定好 python 環境

打開設定環境變數的工具

找到並準備編輯 Path

找到系統的 python，任何 python 版本都可以，複製其資料夾位置 (例如 C://Python27/)

將這幾個資料夾位置合併到原來的 path 參數後面，用分號分隔

貼到 Path 上，設定完成

開啓最基本的 server

打開 cmd

移動到 sigma js 輸出的位置。直接用 F: 之類來更換分隔槽 (從 C: 切換到 F:)，用 cd 資料夾相對位置移動到 sigmajs 輸出的資料夾。 輸入 python -m SimpleHTTPServer

到 http://localhost:8000 看成果
