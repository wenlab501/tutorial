安裝 multimode 轉換 套件

因爲 Netvizz 抓下來的資料是 two-mode 資料 -- po文:回覆者 -- 所以需要裝一個可以作 two-mode 轉 one-mode 的套件： Multimode Networks 以下簡單說明安裝這套件部分，其他部分請參考 ： 下載 Gephi 及 安裝 Gephi。

打開 Plugin 管理頁面



在Available Plugins 頁籤下，選擇 MultimodeNetworksTransformation，按 Install 開始安裝



可以到這 plugin 的 github 頁面看看基本說明： Multimode Networks
