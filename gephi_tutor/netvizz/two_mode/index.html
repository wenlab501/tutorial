2-mode 網絡操作

初步先對 2-mode 的 粉專網絡做一些觀察。

匯入網路資料

File --> 選擇 .gdf 檔 --> Open



將網絡類型改成 Undirected， 這是因爲後面要用 Multimode Networks Transformation 工具轉換網路，而轉換後是無方向網路，所以這裏先將網絡設定成無向(Undirected)。點 OK 將網路匯入 Gephi



匯入後頁面如下，這是幾乎所有網路檔案第一次被放進 Gephi 的樣子



基本網絡設計

開始做一些 Layout 的調整，細節請參考 網絡的佈局



調整顏色，細節請參考 網絡節點顏色。這邊先用 type 作爲分顏色的基礎，將節點分成 post 及 user 兩類



調整大小，細節請參考 網絡節點的大小



基本網絡分析

初步跑一個 模組化分析，其他分析、操作請參考： 網絡分析，網絡過濾

舉個例子，進行 Network Diameter 分析，計算網絡的直徑。因爲這過程需要找到所節點配對之間的最短路徑(shortest path)，而這也可以進一步計算 Betweenness 及 Closeness 等節點的指標，所以 Gephi 也會順道將結果記錄在節點的表單上。
